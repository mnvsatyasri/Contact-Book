# Contact-Book

A simple command-line Contact Book built in Python using the `csv` module.  
You can **add**, **view**, and **search** contacts easily, with data stored in a CSV file.

## Features
- Add new contacts (name + phone number)
- View all saved contacts
- Search for a contact by name
- Data stored in `Contacts.csv`
## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com
   ```

2. Navigate to the project folder:
   ```bash
   cd Contact-Book
   ```

3. Run the script:
   ```bash
   python contact_book.py
   ```

## Example Usage

```text
Contact Book
1. Add Contacts
2. View Contacts
3. Search Contacts
4. Exit

Choose an option: 1
Enter name: Sri
Enter phone number: 9876543210
Contact added!
```
